# Shootlog - Carnet de Tir Numérique

Ce dépôt contient le code du **Carnet de Tir Numérique** de [tireur.org](https://www.tireur.org/carnet-de-tir.php).

## Fonctionnalités
- Enregistrement des armes, des séances et des opérations d'entretien.
- Tracé interactif des impacts sur cible avec calcul automatique de dispersion (Extreme Spread, MOA, MRAD, MPI).
- Génération vectorielle SVG haute qualité basée sur le générateur de cibles officiel (ISSF, Biathlon, silhouettes, etc.).
- Fonctionnement 100% hors-ligne et local (LocalStorage).
- Export et import des données au format JSON.
- Fiches d'impression adaptées (vierges ou pré-remplies).
